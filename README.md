# 💎 HabitGlass
A premium, minimal, and futuristic Habit Tracker dashboard with a stunning **Glassmorphism UI**.

## ✨ Features
- **Modern Glassmorphism**: High-performance blur and translucent panels.
- **Dynamic Grid**: Spreadsheet-style tracking with weekday orientation.
- **Custom Frequency**: Set habits for Daily, Weekdays, or Weekends.
- **Smart Stats**: Real-time progress bars and streak calculation.
- **Visual Trends**: Monthly completion graphs powered by Chart.js.
- **Dark/Light Mode**: Seamless theme toggling with preference saving.

## 🛠️ Tech Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Chart.js
- LocalStorage
