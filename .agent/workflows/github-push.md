---
description: How to push your local project to GitHub
---

1. **Copy the Repository URL** from your GitHub project page.
2. **Tell Antigravity** the URL and your choice:
   - "Push the whole `kodnest` folder to `[URL]`"
   - "Use name: `[Your Name]` and email: `[Your Email]`"
3. **Approve the Plan** that Antigravity provides.
4. **Watch the Magic**: Antigravity will git init, add, commit, and push for you.
